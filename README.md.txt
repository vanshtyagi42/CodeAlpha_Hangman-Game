# Hangman Game - Professional Edition #

## Project Overview

This is a text-based Hangman Game developed in Python. The player must guess a hidden word one letter at a time before running out of lives. The game uses a predefined list of words and provides feedback on correct and incorrect guesses. This project demonstrates the use of Python fundamentals including loops, conditional statements, strings, lists, sets, functions, and randomization.

## Features

- Random word selection from a predefined list
- Maximum of 6 incorrect guesses allowed
- Input validation for user guesses
- Tracks correct and incorrect guesses
- Displays remaining lives
- Win and loss detection
- Replay option after each game
- Clean and structured code using functions

## Game Rules:

1. A random word is selected.
2. The player guesses one letter at a time.
3. Correct guesses reveal letters in the word.
4. Incorrect guesses reduce the available lives.
5. The player loses after 6 incorrect guesses.
6. The player wins by guessing all letters before running out of lives.
